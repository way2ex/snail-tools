# Snail-Tools
Snail is collection of some tools to generate some common files，such as `package.json`、`LICENSE` and so on.

## Quickstart
Install globally:
```bash
yarn add -g snail-tools
# 或者 npm i -g snail-tools

cd <your-repo-location>
# 然后可以直接使用其中的命令
snail init-pkg
```

## Files that can be generated

- `package.json`
- `LICENSE`
